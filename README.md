# IIITN-Resell

A college marketplace platform for IIIT Nagpur, built as a distributed
Spring Boot microservices system on top of Kafka, RabbitMQ, Postgres,
Redis, and Elasticsearch.

## Status
- [x] Infra (docker-compose)
- [x] Maven parent + shared common module
- [ ] Service discovery + config server
- [ ] API gateway
- [ ] Auth service
- [ ] User service
- [ ] Product service
- [ ] Chat service
- [ ] Notification service
- [ ] Search service
- [ ] Analytics / warehouse service

See the root for the backend modules, `client/` for the React frontend.
